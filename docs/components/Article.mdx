# Article

Use documentation style everywhere.

<carbon-ad />

## Usage

`Article` contains all styles to format documentation pages.

```js noInline
import { Article } from 'smooth-doc/components'
import GettingStarted from './partials/getting-started.mdx'

render(
  <Article>
    <GettingStarted />
  </Article>,
)
```
